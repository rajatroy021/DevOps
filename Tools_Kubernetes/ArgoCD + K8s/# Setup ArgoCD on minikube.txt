# Setup ArgoCD on minikube 
https://argo-cd.readthedocs.io/en/stable/getting_started/


First start Minikube using command ::

start minikube 

kubectl create namespace argocd


kubectl apply -n argocd -f https://raw.githubusercontent.com/argoproj/argo-cd/stable/manifests/install.yaml 


kubectl get pod -n argocd           # Check pod status if running or not 


kubectl port-forward -n argocd svc/argocd-server 8080:443   # Forward the service rqst to local port so that we accss it on our browser 





Start Fresh Cluster - Minikube 
# Delete all deploys on the cluster using command 
minikube delete --all 

# Reset ur cluster back :
Select reset cluster in locker Desktop and restart the application .

# To install the latest minikube stable release on ARM64 macOS using binary download:- Command 

curl -LO https://storage.googleapis.com/minikube/releases/latest/minikube-darwin-arm64
sudo install minikube-darwin-arm64 /usr/local/bin/minikube

minikube start 

kubectl get po -A       	# Interact with your cluster

minikube dashboard - Create dashboard if needed
